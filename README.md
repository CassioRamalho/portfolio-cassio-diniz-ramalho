# Portfólio Cássio Diniz Ramalho

Portfólio pessoal criado para apresentar experiência em dados, BI, automação de processos e projetos de tecnologia.

## Estrutura

- `index.html`: página principal do portfólio.
- `style.css`: visual, responsividade e animações leves.
- `script.js`: menu mobile, reveal on scroll e carrosséis dos cases.
- `assets/docs/`: currículo em PDF.
- `assets/images/`: capturas dos projetos em destaque.
- `assets/icons/favicon.svg`: ícone do site.

## Publicação

Por ser um site estático, funciona bem em:

- GitHub Pages
- Netlify
- Vercel
- qualquer hospedagem tradicional com HTML/CSS/JS

## Observação

Os links de demo ao vivo apontam para as páginas públicas atuais da CDR quando disponíveis. O restante do portfólio foi estruturado para funcionar de forma independente.
